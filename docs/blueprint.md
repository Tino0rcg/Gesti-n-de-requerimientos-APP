# **App Name**: ServiDesk

## Core Features:

- Ticket Submission Form: Allows users to submit new support requests, including fields for subject, detailed description, priority, and relevant attachments.
- Ticket Management Dashboard: A centralized dashboard for IT staff to view all submitted tickets, with options to filter by status, priority, agent, and category.
- Detailed Ticket View: Displays the full details of a specific ticket, including history, notes, and the ability for IT staff to update status (Open, In Progress, Resolved, Closed).
- Agent Assignment & Notes: Enables IT managers or leads to assign tickets to specific IT support agents and for agents to add internal notes.
- User Authentication & Roles: Secure login system with distinct roles for general users (submitters) and IT staff (agents/managers) with appropriate access permissions.
- AI-Powered Categorization Tool: An AI tool that analyzes the ticket description and automatically suggests an appropriate category (e.g., Network, Software, Hardware) and priority level.
- Basic Reporting & Metrics: Provides a simple overview of key metrics, such as the number of open tickets, resolved tickets, and average resolution time, for general insight.

## Style Guidelines:

- Primary color: A deep, professional blue (#1F4793) to convey trust and reliability, fitting for an IT service application.
- Background color: A very light, subtly blue-tinged grey (#ECEFF3) to maintain a clean, uncluttered, and efficient feel in the user interface.
- Accent color: A vibrant yet calm turquoise (#28C0E6) used for interactive elements like buttons, links, and highlighted status indicators to draw attention effectively.
- Body and headline font: 'Inter' (sans-serif) for its modern, objective, and highly legible characteristics, suitable for conveying information clearly in a professional environment.
- Utilize clean, modern line icons for actions, categories, and status indicators. Icons should be easily understandable at a glance to enhance user navigation.
- Implement a clear, spacious layout with a fixed sidebar for primary navigation and a content area for forms, tables, and dashboards, optimizing for desktop usability.
- Subtle and functional animations, such as smooth transitions for content loading, filter changes, and form submissions, providing clear visual feedback without being distracting.
